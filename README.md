# Klasifikasi Kualitas Anggur Menggunakan Machine Learning
Proyek ini bertujuan untuk memprediksi kualitas anggur menggunakan beberapa algoritma machine learning seperti Logistic Regression, KNN, Decision Tree, dan Random Forest.
Tahapan yang dilakukan meliputi:
- Exploratory Data Analysis (EDA)
- Preprocessing Data
- Training Model
- Evaluasi Model
- Prediksi Data Testing
Model terbaik digunakan untuk menghasilkan file prediksi akhir dalam format CSV.
